## 1. Getting Started

### Logging In

1. Go to `https://app.lineup.com`
2. Enter your email and password
3. Complete 2FA if enabled
4. You'll land on the Dashboard

### First-Time Setup

1. **Connect your calendar** (Google or Outlook) for scheduling
2. **Set your availability** in Calendar settings
3. **Customize notifications** in Settings

### Navigation

| Menu Item | Purpose |
|-----------|---------|
| Dashboard | Overview and quick stats |
| Candidates | Manage candidate profiles |
| Interviews | View and schedule interviews |
| Calendar | Calendar view of all events |
| Reports | Analytics and metrics |

## 3. Managing Candidates

### Adding a Single Candidate

1. Click **Candidates** → **Add Candidate**
2. Fill in:
   - Name (required)
   - Email
   - Phone
   - Role/Position applying for
   - Source (LinkedIn, Referral, etc.)
3. Upload resume (optional)
4. Click **Save**

### Bulk Import Candidates

1. Go to **Candidates** → **Import**
2. Choose import method:
   - **CSV Upload**: Download template, fill in, upload
   - **Resume Parsing**: Upload multiple resumes
3. Review parsed data
4. Click **Import All** or select specific candidates

### Resume Parsing

Lineup automatically extracts:
- Name, email, phone
- Skills and experience
- Education
- Work history

Review and edit before saving.

### Searching & Filtering

Use filters to find candidates:

| Filter | Options |
|--------|---------|
| Stage | Applied, Screening, Interview, Offer, Hired, Rejected |
| Source | LinkedIn, Referral, Website, etc. |
| Role | Filter by position |
| Date | Date range |
| Tags | Custom tags |

### Candidate Profile

Click any candidate to view:

- **Overview**: Contact info, current stage, scores
- **Timeline**: Activity history
- **Interviews**: Scheduled and completed
- **Documents**: Resumes, portfolios
- **Notes**: Team notes and comments
- **Feedback**: Interview feedback summary

### Candidate Actions

| Action | How |
|--------|-----|
| Change Stage | Click stage dropdown → Select new stage |
| Schedule Interview | Click **Schedule Interview** button |
| Add Note | Notes tab → Add Note |
| Upload Document | Documents tab → Upload |
| Send Message | Click email/SMS icon |
| Archive | More menu → Archive |

## 4. Scheduling Interviews

### Quick Schedule

1. From candidate profile, click **Schedule Interview**
2. Select:
   - Interview type (Phone, Video, In-person)
   - Interviewer(s)
   - Date and time
   - Duration
3. Add meeting link (or auto-generate)
4. Click **Schedule**

### Calendar View Scheduling

1. Go to **Calendar**
2. Click on desired time slot
3. Select **New Interview**
4. Search and select candidate
5. Complete interview details
6. Click **Save**

### Bulk Scheduling

For multiple interviews:

1. Go to **Interviews** → **Bulk Schedule**
2. Select candidates
3. Set interview parameters
4. System suggests optimal times based on availability
5. Review and confirm

### Rescheduling

1. Open interview details
2. Click **Reschedule**
3. Select new date/time
4. Notifications sent automatically to all participants

### Cancelling

1. Open interview details
2. Click **Cancel Interview**
3. Select reason
4. Confirm cancellation
5. Notifications sent to participants

## 5. Interview Feedback

### Submitting Feedback

After an interview:

1. Go to **Interviews** → Select completed interview
2. Or access via Dashboard → **Pending Feedback**
3. Fill in:
   - **Overall Rating**: 1-5 stars
   - **Technical Score**: (if applicable)
   - **Communication Score**: 1-5
   - **Culture Fit Score**: 1-5
   - **Strengths**: Key positives
   - **Concerns**: Areas of concern
   - **Recommendation**: Strong Hire / Hire / No Hire / Strong No Hire
   - **Notes**: Detailed comments
4. Click **Submit Feedback**

### Viewing Team Feedback

1. Open candidate profile
2. Go to **Feedback** tab
3. View all feedback from team members
4. See aggregated scores

### Feedback Best Practices

- Submit feedback within 24 hours
- Be specific and objective
- Include examples from the interview
- Avoid bias-prone language

## 6. Pipeline Management

### Hiring Stages

Default pipeline stages:

1. **Applied**: New applications
2. **Screening**: Resume/phone screening
3. **Interview**: In interview process
4. **Offer**: Offer extended
5. **Hired**: Accepted and hired
6. **Rejected**: Not moving forward

### Moving Candidates

**Single candidate:**
1. In candidate list or profile
2. Click stage dropdown
3. Select new stage
4. Add reason (optional)
5. Confirm

**Bulk move:**
1. Select multiple candidates (checkboxes)
2. Click **Bulk Actions** → **Change Stage**
3. Select target stage
4. Confirm

### Stage Automation

Automations may trigger:
- Email to candidate (e.g., rejection notice)
- Notification to hiring team
- Integration sync (e.g., to ATS)
- Task assignment

## 7. Communication

### Email

**Sending email to candidate:**
1. Open candidate profile
2. Click **Email** icon
3. Select or customize template
4. Preview and send

**Templates available:**
- Interview invitation
- Interview reminder
- Follow-up
- Offer letter
- Rejection

### SMS

1. Click **SMS** icon on candidate profile
2. Type or select template
3. Send

### Bulk Communication

1. Select multiple candidates
2. Click **Bulk Actions** → **Send Email** or **Send SMS**
3. Choose template
4. Personalization variables auto-filled
5. Send

### Communication History

View all communications in:
- Candidate profile → Timeline
- Communication → History

## 8. Reports & Analytics

### Available Reports

| Report | Description |
|--------|-------------|
| Hiring Funnel | Candidates by stage |
| Time to Hire | Days from applied to hired |
| Source Analysis | Performance by source |
| Interviewer Load | Interviews per team member |
| Stage Metrics | Time spent in each stage |

### Generating Reports

1. Go to **Reports**
2. Select report type
3. Set filters (date range, role, team)
4. Click **Generate**

### Exporting

Export options:
- **PDF**: Formatted report
- **CSV**: Raw data for analysis
- **Excel**: Spreadsheet format

Click **Export** and select format.

### Scheduled Reports

1. Go to **Reports** → **Scheduled**
2. Click **Create Schedule**
3. Configure:
   - Report type
   - Frequency (daily, weekly, monthly)
   - Recipients
   - Format
4. Save

Reports delivered via email automatically.

## 9. Calendar Integration

### Connecting Calendar

1. Go to **Settings** → **Calendar**
2. Click **Connect Google Calendar** or **Connect Outlook**
3. Authorize access
4. Select calendars to sync

### Two-Way Sync

Connected calendars provide:
- See all events in Lineup calendar
- New interviews sync to your calendar
- External events block availability
- Changes sync both directions

### Availability Settings

1. Go to **Settings** → **Availability**
2. Set:
   - Working hours
   - Buffer between meetings
   - Minimum notice for scheduling
3. Block out recurring unavailable times

### Video Meeting Links

Auto-generate meeting links:
- Google Meet (with Google Calendar)
- Microsoft Teams (with Outlook)
- Custom link (Zoom, etc.)

## 10. Tips & Best Practices

### Efficiency Tips

1. **Use keyboard shortcuts**:
   - `Cmd/Ctrl + K`: Quick search
   - `N`: New candidate
   - `I`: New interview

2. **Set up saved filters** for common searches

3. **Use templates** for consistent communication

4. **Enable notifications** for important events

### Collaboration

- Add notes after every interaction
- @mention team members in notes
- Use tags to categorize candidates
- Share candidate profiles with hiring managers

### Data Quality

- Keep candidate information up-to-date
- Merge duplicate profiles
- Archive old candidates
- Use consistent naming for sources/tags

### Compliance

- Document all hiring decisions
- Submit feedback promptly
- Follow company interview guidelines
- Maintain candidate confidentiality
