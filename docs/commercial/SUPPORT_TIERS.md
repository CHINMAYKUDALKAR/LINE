## Priority Definitions

### P1 - Critical

**Definition:** Complete service outage or severe security incident

**Examples:**
- System completely unavailable
- Data breach or security vulnerability
- Data loss or corruption
- All users affected

**Expected Resolution:** 2-4 hours

---

### P2 - High

**Definition:** Major functionality severely impaired

**Examples:**
- Core feature (scheduling, candidates) non-functional
- Integration sync completely failing
- Significant performance degradation (>5s response times)
- Major subset of users affected

**Expected Resolution:** 4-8 hours

---

### P3 - Medium

**Definition:** Feature partially impaired, workaround available

**Examples:**
- Non-critical feature bug
- Intermittent issues
- UI/UX problems
- Minor functionality gap

**Expected Resolution:** 1-3 business days

---

### P4 - Low

**Definition:** Minor issues with minimal business impact

**Examples:**
- Documentation questions
- Feature requests
- Minor cosmetic issues
- How-to questions

**Expected Resolution:** 3-5 business days

## Support Best Practices

### When Contacting Support

Include:
1. **Clear subject line** - Brief description
2. **Steps to reproduce** - What you did
3. **Expected vs. actual** - What should happen vs. what happened
4. **Screenshots/videos** - Visual evidence
5. **Environment info** - Browser, OS
6. **Impact** - Who/what is affected

### Example Ticket

```
Subject: Interview scheduling fails with Google Calendar

Priority: P2 - High

Description:
When scheduling an interview, the event is not syncing
to Google Calendar.

Steps to reproduce:
1. Go to Candidates > Select candidate
2. Click "Schedule Interview"
3. Fill in details, click Save
4. Check Google Calendar - event not present

Expected: Event appears in Google Calendar
Actual: Event saves in Lineup but doesn't sync

Error message: "Sync failed: 403 Forbidden"

Impact: 10+ interviews pending, affecting 3 recruiters

Browser: Chrome 120.0
Environment: Production
```
