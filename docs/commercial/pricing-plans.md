## Pricing Dimensions

### Seat-Based
- Each named user counts as a seat
- Recruiters, Hiring Managers, Admins

### Usage-Based (Tracked)
- **Active recruiters/month** - Users who log in
- **Candidates created** - New candidates added
- **Interviews scheduled** - Interview events created
- **Messages sent** - Email, SMS, WhatsApp volume

### Not Charged
- Interview reschedules
- Candidate stage changes
- Feedback submissions
- Report generation

## Overage Policy

When soft limits are exceeded:

1. **No blocking** - Service continues
2. **Notification** - Admin notified at 80%, 100%
3. **Review** - Discuss plan upgrade at next renewal
4. **Enforcement** - None in v1 (measurement only)
